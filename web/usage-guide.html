使用说明

3分钟快速上手

准备工作

将设备插入电脑USB口，确保周围有可用的WiFi网络（2.4GHz）。

烧录固件
通过浏览器直接烧录，无需安装任何软件
⚠️ 进入下载模式（重要）

1. 按住 BOOT 按钮 不松手
2. 按下 RST 按钮（复位）
3. 松开 RST 按钮
4. 松开 BOOT 按钮
此时设备进入下载模式，等待浏览器连接

✅ 需要 Chrome/Edge 浏览器(89+版本)

弹窗英文说明：
• Connect = 连接设备
• Install = 安装固件
• Success! = 烧录成功
使用步骤
1
首次配网

设备首次启动会进入配网模式：

操作示例
打开手机WiFi → 连接热点 WOL-Mini-USB → 浏览器自动弹出配网页面
2
设置WiFi和密码
操作示例
点击"扫描WiFi" → 选择 MyHome-WiFi → 输入密码 12345678
注意
仅支持2.4GHz WiFi，不支持5GHz
3
设置设备ID和Token
配置示例
设备名称：mypc（随便起名）
设备ID：mypc-150C（自动生成，记住这个）
Token：abcd1234（自己设置，记住这个）
重要
设备ID和Token要记住，远程控制时需要输入
4
开始远程控制
操作示例
打开控制网页 → 输入设备ID mypc-150C → 输入Token abcd1234 → 点击"唤醒电脑"
LED状态说明
闪烁配网模式（等待配网）
常亮WiFi已连，MQTT未连接
常灭正常工作状态
常见问题
Q: 电脑关机了能唤醒吗？
不能，只能唤醒睡眠状态的电脑。
Q: 忘记Token怎么办？
长按BOOT按钮5秒进入配网模式，可查看和修改Token（不会清空原有配置）。
Q: 不在同一WiFi能用吗？
可以，远程控制通过互联网，不在同一城市也能用。
Q: 安全吗？
Token是安全密码，只有知道Token的人才能控制。
